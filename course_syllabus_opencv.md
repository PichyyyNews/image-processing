## 1. คำอธิบายรายวิชา (Course Description)
ศึกษาและปฏิบัติเกี่ยวกับหลักการ ทฤษฎีภาพดิจิทัล การประมวลผลภาพเชิงดิจิทัลเบื้องต้น การปรับปรุงคุณภาพภาพในโดเมนเชิงพื้นที่ การปรับปรุงภาพในโดเมนเชิงความถี่ การซ่อมคืนสภาพภาพ การประมวลผลภาพสีในระดับพิกเซล การบีบอัดภาพ การแบ่งส่วนภาพ การหาขอบหาจุดเด่นภายในภาพ และการจับความเคลื่อนไหว

## 2. ผลลัพธ์การเรียนรู้ระดับรายวิชา (Course Learning Outcomes - CLO)
1. **CLO 1**: สามารถอธิบายหลักการ ทฤษฎีพื้นฐานของภาพดิจิทัล และคณิตศาสตร์ที่เกี่ยวข้องกับการประมวลผลภาพได้
2. **CLO 2**: พัฒนาทักษะการเขียนโปรแกรมภาษา Python ร่วมกับ OpenCV ในการประมวลผลภาพดิจิทัล ปรับปรุงคุณภาพภาพ แบ่งส่วนภาพ และสกัดคุณลักษณะเด่นของภาพได้
3. **CLO 3**: สามารถวิเคราะห์ เลือกใช้ และประยุกต์ใช้อัลกอริทึมในการประมวลผลภาพและคอมพิวเตอร์วิทัศน์ที่เหมาะสมกับโจทย์ปัญหาในชีวิตจริง
4. **CLO 4**: ปฏิบัติงานด้วยความประณีต รอบคอบ และปลอดภัย ตระหนักถึงจริยธรรมการใช้งานข้อมูลภาพและปัญญาประดิษฐ์ในงานอาชีพ

## 4. แผนการจัดหน่วยเรียนรู้รายสัปดาห์ (15-Week Detailed Schedule)
จัดการเรียนการสอน 5 ชั่วโมงต่อสัปดาห์ (บรรยาย 2 ชม. + ปฏิบัติ 3 ชม.)

### สัปดาห์ที่ 1: บทนำสู่การประมวลผลภาพดิจิทัลและสภาพแวดล้อมการพัฒนา
*   **บรรยาย (2 ชั่วโมง)**:
    *   ประวัติศาสตร์ เทรนด์ปัจจุบัน และกรณีการใช้งานจริงของการประมวลผลภาพดิจิทัล (DIP) และคอมพิวเตอร์วิทัศน์ (Computer Vision)
    *   โครงสร้างของภาพดิจิทัล: พิกเซล (Pixels), ความละเอียด (Resolution), ความลึกบิต (Bit Depth), ระบบพิกัดภาพ (Image Coordinate System)
    *   ความแตกต่างระหว่างภาพขาวดำ (Binary), ภาพเกรย์สเกล (Grayscale) และภาพสี (RGB/BGR Matrix)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 1: Environment Setup & Basic Image I/O**:
    *   การติดตั้ง Anaconda, VS Code และการจัดการสภาพแวดล้อมเสมือน (Virtual Environment)
    *   การติดตั้งแพ็คเกจที่จำเป็น: `opencv-python`, `numpy`, `matplotlib`
    *   การเขียนโปรแกรมอ่านภาพ (`cv2.imread()`), การตรวจสอบโครงสร้างและมิติของภาพด้วย NumPy (`image.shape`, `image.size`, `image.dtype`)
    *   การแสดงผลภาพด้วยหน้าต่าง OpenCV (`cv2.imshow()`, `cv2.waitKey()`) และ Matplotlib
    *   การบันทึกภาพลงเครื่องคอมพิวเตอร์ (`cv2.imwrite()`)
*   **เครื่องมือหลัก**: Python 3.10+, VS Code, OpenCV (cv2), NumPy, Matplotlib
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.imread()`, `cv2.imshow()`, `cv2.imwrite()`, `cv2.cvtColor()`
*   **งานปฏิบัติ**: ทดลองโหลดรูปภาพสี แปลงเป็นภาพเกรย์สเกล และบันทึกผลลัพธ์เป็นไฟล์ฟอร์แมตต่าง ๆ (.jpg, .png) เพื่อวิเคราะห์ความแตกต่างของขนาดไฟล์และแชนเนลสี

### สัปดาห์ที่ 2: โครงสร้างข้อมูลพิกเซล การเข้าถึงข้อมูลและการดำเนินการเบื้องต้น
*   **บรรยาย (2 ชั่วโมง)**:
    *   การเก็บข้อมูลของ OpenCV: ระบบสี BGR (Blue, Green, Red) ที่แตกต่างจากมาตรฐาน RGB
    *   การเข้าถึงข้อมูลอาเรย์ของรูปภาพโดยใช้ NumPy indexing (Slicing)
    *   แนวคิดเรื่องพื้นที่สนใจเฉพาะส่วน (Region of Interest: ROI) และการประยุกต์ใช้
    *   การดำเนินการทางเรขาคณิตขั้นพื้นฐาน: การย่อ-ขยายภาพ (Resizing), การตัดภาพ (Cropping), การกลับภาพ (Flipping), และการหมุนภาพ (Rotation)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 2: Image Manipulation & ROI Extraction**:
    *   เขียนโปรแกรมเข้าถึงค่าสีพิกเซลในตำแหน่งที่กำหนด (x, y) และแก้ไขค่าสี
    *   การสร้างกรอบพื้นที่สนใจ (ROI) และดึงส่วนนั้นออกมาสร้างเป็นภาพใหม่
    *   การย่อขยายภาพด้วยฟังก์ชัน Interpolation รูปแบบต่าง ๆ (INTER_NEAREST, INTER_LINEAR, INTER_CUBIC)
    *   การทำทรานส์ฟอร์มเบื้องต้น (Affine Transformation): การสร้าง Matrix การหมุนและหมุนภาพ
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.resize()`, `cv2.flip()`, `cv2.getRotationMatrix2D()`, `cv2.warpAffine()`
*   **งานปฏิบัติ**: ให้นักเรียนดึงหน้าของบุคคลในภาพหมู่ (ROI Extraction) จากนั้นนำรูปหน้านั้นมาย่อขยาย และกลับภาพด้านซ้ายขวา ก่อนนำไปแปะทับลงในภาพเดิมแบบสร้างสรรค์

### สัปดาห์ที่ 3: การประมวลผลจุดพิกเซลและการดำเนินการทางคณิตศาสตร์/ตรรกศาสตร์
*   **บรรยาย (2 ชั่วโมง)**:
    *   การดำเนินการระดับจุด (Point Processing)
    *   การปรับปรุงความสว่าง (Brightness) และความคมชัด (Contrast) ของภาพด้วยสมการคณิตศาสตร์เชิงเส้น $g(x,y) = \alpha f(x,y) + \beta$
    *   การประมวลผลเชิงตัวเลขระหว่างสองรูปภาพ: การบวก การลบ การคูณ และการหารภาพ
    *   ปัญหาความล้นค่าพิกเซล (Pixel Overflow) และความสำคัญของฟังก์ชันอิ่มตัว (Saturation Operations)
    *   การดำเนินการเชิงตรรกศาสตร์บนบิตของภาพ (Bitwise Operations: AND, OR, XOR, NOT) และการประยุกต์ทำหน้ากากภาพ (Image Masking)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 3: Image Blending & Masking Application**:
    *   การปรับแต่งแสงเงาแบบ Interactive ด้วย GUI Trackbar ของ OpenCV
    *   การรวมรูปภาพสองรูปเข้าด้วยกันแบบโปร่งแสงโดยใช้การถ่วงน้ำหนัก (Image Blending)
    *   การสร้างหน้ากากภาพทรงกลม/สี่เหลี่ยมเพื่อตัดภาพส่วนที่ต้องการด้วย `cv2.bitwise_and()`
    *   การประยุกต์สร้างระบบ Chroma Key (กรีนสกรีนแบบจำลองระดับพิกเซล)
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.add()`, `cv2.subtract()`, `cv2.addWeighted()`, `cv2.bitwise_and()`, `cv2.bitwise_or()`, `cv2.createTrackbar()`
*   **งานปฏิบัติ**: ออกแบบเครื่องมือสร้างภาพซ้อน (Double Exposure Effect) จากรูปภาพสองภาพ โดยให้ผู้ใช้งานสามารถปรับระดับความโปร่งแสงของการรวมภาพผ่านการสไลด์ Trackbar ได้ในแบบเรียลไทม์

### สัปดาห์ที่ 4: การวิเคราะห์และการปรับปรุงภาพด้วยฮิสโตแกรม
*   **บรรยาย (2 ชั่วโมง)**:
    *   ความรู้เบื้องต้นเกี่ยวกับกราฟฮิสโตแกรมของภาพ (Image Histogram): โทนสว่าง โทนมืด และค่าคอนทราสต์
    *   การวิเคราะห์การกระจายแสงจากกราฟฮิสโตแกรม
    *   การปรับปรุงคุณภาพภาพด้วยการกระจายความเข้มของฮิสโตแกรม (Histogram Equalization)
    *   ข้อจำกัดของการทำ Equalization แบบปกติ และการแก้ไขด้วย Adaptive Histogram Equalization (CLAHE: Contrast Limited Adaptive Histogram Equalization)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 4: Contrast Enhancement and CLAHE**:
    *   การพล็อตกราฟฮิสโตแกรมของภาพขาวดำและภาพสี BGR แยกแต่ละแชนเนลสีด้วย Matplotlib
    *   การทดลองใช้ฟังก์ชัน `cv2.equalizeHist()` ในภาพขาวดำ
    *   การนำเทคนิค Equalization ไปใช้กับภาพสีโดยต้องแปลงภาพไปยังระบบสี YCrCb หรือ HSV เสียก่อน
    *   การประยุกต์ใช้งาน CLAHE เพื่อแก้ไขภาพถ่ายย้อนแสงหรือภาพเอกซเรย์แพทย์ที่มีแสงไม่เท่ากัน
*   **เครื่องมือหลัก**: Python, OpenCV, Matplotlib
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.calcHist()`, `cv2.equalizeHist()`, `cv2.createCLAHE()`
*   **งานปฏิบัติ**: เขียนโปรแกรมเปรียบเทียบคุณภาพของรูปภาพที่ได้จาก 3 วิธีการ: 1) ภาพต้นฉบับ 2) ภาพหลังปรับ Equalization ปกติ 3) ภาพหลังปรับด้วย CLAHE พร้อมพล็อตฮิสโตแกรมของทั้ง 3 แบบเปรียบเทียบกันในหน้าต่างเดียว

### สัปดาห์ที่ 5: การปรับปรุงภาพในโดเมนเชิงพื้นที่ด้วยการกรองภาพและการลดสัญญาณรบกวน
*   **บรรยาย (2 ชั่วโมง)**:
    *   แนวคิดการกรองภาพเชิงพื้นที่ (Spatial Filtering) และการสังเคราะห์พื้นที่ข้างเคียง (Neighborhood Processing)
    *   หลักการทำ Convolution ของตัวกรอง (Kernel/Filter Matrix)
    *   ความเข้าใจทฤษฎีกล่องเบลอ (Averaging / Box Filter), ตัวกรองแบบเกาส์เซียน (Gaussian Blur)
    *   การลบสัญญาณรบกวน (Noise): สัญญาณรบกวนแบบเกาส์เซียน และแบบเกลือพริกไทย (Salt & Pepper Noise)
    *   ความแตกต่างและประสิทธิภาพของตัวกรองมัธยฐาน (Median Filter) และตัวกรองแบบทวิภาคี (Bilateral Filter - รักษาขอบคม)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 5: Image Smoothing and Noise Reduction**:
    *   การประมวลผลฟิลเตอร์แบบกำหนด Kernel เองด้วยการคำนวณผ่าน `cv2.filter2D()`
    *   การทดลองใส่สัญญาณรบกวนจำลองเข้าไปในภาพ และใช้ Filter รูปแบบต่าง ๆ ในการกู้คืนคุณภาพ
    *   การศึกษาเปรียบเทียบผลลัพธ์ของ `cv2.blur()`, `cv2.GaussianBlur()`, `cv2.medianBlur()`, และ `cv2.bilateralFilter()`
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.filter2D()`, `cv2.blur()`, `cv2.GaussianBlur()`, `cv2.medianBlur()`, `cv2.bilateralFilter()`
*   **งานปฏิบัติ**: เขียนระบบฟิลเตอร์กล้องยอดนิยม เช่น สร้างเอฟเฟกต์เบลอภาพพื้นหลังโดยรักษาความคมชัดของรายละเอียดขอบวัตถุไว้ (Cartoonization preprocessing)

### สัปดาห์ที่ 6: การตรวจหาขอบและการสกัดเส้นขอบในภาพ (Edge Detection & Gradients)
*   **บรรยาย (2 ชั่วโมง)**:
    *   ความเข้าใจเกี่ยวกับการหาอนุพันธ์ของภาพภาพ (Image Gradient)
    *   ตัวดำเนินการหาอนุพันธ์เชิงพื้นที่อันดับหนึ่ง: Sobel Operator และ Scharr Operator
    *   ตัวดำเนินการหาอนุพันธ์เชิงพื้นที่อันดับสอง: Laplacian Operator
    *   การวิเคราะห์สัญญาณและการสกัดขอบขั้นสูงด้วยอัลกอริทึม Canny Edge Detection (Non-maximum suppression และ Hysteresis thresholding)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 6: Edge Detector and Document Scanner Preprocessing**:
    *   การเขียนโปรแกรมตรวจจับขอบของภาพในแกน X และแกน Y ด้วย Sobel พร้อมการผสานภาพขอบ
    *   การตรวจหาขอบภาพความไวสูงด้วย Laplacian
    *   การเขียนแอปพลิเคชันเล็กสำหรับทดลองเปลี่ยนค่าขีดจำกัดต่ำและสูง (Min/Max Thresholds) ของ Canny Edge Detection ด้วย Trackbar แบบโต้ตอบได้
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.Sobel()`, `cv2.Scharr()`, `cv2.Laplacian()`, `cv2.Canny()`

### สัปดาห์ที่ 7: การประมวลผลภาพเชิงสัณฐานวิทยาและการวิเคราะห์รูปทรงเบื้องต้น
*   **บรรยาย (2 ชั่วโมง)**:
    *   ทฤษฎีการประมวลผลภาพเชิงสัณฐานวิทยา (Morphological Image Processing)
    *   การดำเนินการพื้นฐาน: การกร่อนภาพ (Erosion) และการแผ่ขยายภาพ (Dilation)
    *   การดำเนินการขั้นสูง: การเปิดภาพ (Opening - ลบจุดรบกวน), การปิดภาพ (Closing - เชื่อมช่องว่าง), การหาเส้นขอบทางสัณฐานวิทยา (Morphological Gradient), Top Hat และ Black Hat
    *   แนวคิดการหาโครงร่างภายนอก (Contour Detection) ของรูปภาพ
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 7: Object Counting & Contour Analysis**:

    *   การเขียนโปรแกรมตรวจจับโครงร่างด้วย `cv2.findContours()`
    *   การสกัดคุณลักษณะของเส้นขอบ: พื้นที่ (Area), ความยาวรอบรูป (Perimeter), จุดศูนย์กลางมวล (Moments/Centroid) และการวาดกล่องตีกรอบวัตถุ (Bounding Box)
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.morphologyEx()`, `cv2.findContours()`, `cv2.drawContours()`, `cv2.moments()`, `cv2.boundingRect()`, `cv2.minAreaRect()`
*   **งานปฏิบัติ**: ออกแบบระบบนับจำนวนวัตถุอัตโนมัติ (เช่น นับเม็ดยา หรือเหรียญในภาพ) โดยการกรอง Noise, ทำ Morphological operations เพื่อแยกเม็ดยาที่เบียดกัน, จากนั้นหาเส้นขอบ แสดงตัวเลขนับจำนวน และวาดกรอบสีล้อมรอบเหรียญแต่ละเหรียญที่นับได้

### สัปดาห์ที่ 10: การแบ่งส่วนภาพขั้นสูงและการค้นหาจุดเด่นเด่นของภาพ (Segmentation & Features)
*   **บรรยาย (2 ชั่วโมง)**:
    *   การแบ่งส่วนภาพขั้นสูงอิงพื้นที่และขอบเขต: Watershed Algorithm และการหาเส้นขอบแบ่งสิ่งกีดขวาง
    *   การแบ่งส่วนภาพแบบมีปัญญาประดิษฐ์และกึ่งอัตโนมัติ: GrabCut Algorithm
    *   ความเข้าใจเรื่องจุดสนใจเด่นในภาพ (Features / Keypoints)
    *   อัลกอริทึมการสกัดมุมและจุดเด่นหลัก: Harris Corner Detection, SIFT (Scale-Invariant Feature Transform) และ ORB (Oriented FAST and Rotated BRIEF)
    *   การจับคู่ฟีเจอร์ของภาพและการเปรียบเทียบความคล้าย (Feature Matching)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 10: Image Stitching and Object Matching**:
    *   การประยุกต์ใช้ Watershed Algorithm เพื่อแยกภาพวัตถุที่ติดกันเหนียวแน่น
    *   การสกัดจุด Keypoints และหาตัวระบุคุณลักษณะเด่น (Descriptors) ด้วย ORB
    *   การเขียนโปรแกรมจับคู่รูปภาพ 2 รูปที่มีมุมมองและองศาต่างกันโดยใช้ Matcher (Brute-Force Matcher หรือ FLANN Matcher)
    *   การนำเสนอตัวอย่างโปรแกรมสร้างภาพพาโนรามา (Image Stitching) เบื้องต้น
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.watershed()`, `cv2.grabCut()`, `cv2.cornerHarris()`, `cv2.ORB_create()`, `cv2.BFMatcher()`
*   **งานปฏิบัติ**: เขียนโปรแกรมสแกนโลโก้หรือค้นหาสิ่งของต้นแบบในภาพถ่ายที่มีการบิดเบี้ยวและเปลี่ยนขนาด (เช่น ค้นหาปกหนังสือต้นฉบับในกองหนังสือเบี้ยว ๆ) โดยใช้การจับคู่คุณลักษณะแบบ ORB และคำนวณหา Homography เพื่อทำ Perspective Transformation ล้อมกรอบหนังสือเล่มนั้น

### สัปดาห์ที่ 11: การซ่อมคืนสภาพภาพและการบีบอัดภาพดิจิทัล
*   **บรรยาย (2 ชั่วโมง)**:
    *   แบบจำลองความเสื่อมโทรมและการซ่อมคืนสภาพของภาพ (Image Degradation and Restoration Model)
    *   ทฤษฎีการลบรอยขีดข่วน คราบ หรือสิ่งแปลกปลอมในรูปภาพ (Image Inpainting)
    *   ทฤษฎีการบีบอัดภาพ (Image Compression): แบบสูญเสียข้อมูลบางส่วน (Lossy) และแบบไม่สูญเสียข้อมูล (Lossless)
    *   หลักการทำงานของฟอร์แมตไฟล์ JPEG และ PNG (การทำงานของ DCT - Discrete Cosine Transform)
    *   ตัววัดประเมินความคมชัดของภาพที่สร้างใหม่: Mean Squared Error (MSE) และ Peak Signal-to-Noise Ratio (PSNR)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 11: Inpainting & Compression Quality Test**:
    *   การซ่อมแซมภาพเก่าโบราณที่มีริ้วรอยขาดชำรุดด้วยฟังก์ชัน `cv2.inpaint()` (การเปรียบเทียบระหว่างวิธี Navier-Stokes และวิธี Fast Marching)
    *   การทดลองบีบอัดภาพแบบปรับคุณภาพเปอร์เซ็นต์ (JPEG Compression Quality) และคำนวณเปรียบเทียบค่าความผิดเพี้ยนของภาพด้วย MSE และ PSNR ใน Python
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.inpaint()`, `cv2.imencode()`
*   **งานปฏิบัติ**: นำภาพประวัติศาสตร์ที่มีรอยปากกาขีดเขียนทับมาทำความสะอาด โดยนักศึกษาต้องสร้าง Mask ภาพรอยขีดข่วนขึ้นมาก่อน (ผ่านการสเกลสีหรือ Morphological) จากนั้นป้อนข้อมูลให้ `cv2.inpaint()` ซ่อมแซมรูปภาพให้สวยงามเป็นธรรมชาติ พร้อมวัดค่าคุณภาพผลลัพธ์เชิงตัวเลข

### สัปดาห์ที่ 12: การปรับปรุงภาพในโดเมนเชิงความถี่
*   **บรรยาย (2 ชั่วโมง)**:
    *   ความแตกต่างระหว่างพื้นที่เชิงเวลา/พื้นที่ (Spatial Domain) และเชิงความถี่ (Frequency Domain)
    *   การแปลงฟูเรียร์แบบดีสครีตแบบสองมิติ (2D Discrete Fourier Transform - DFT)
    *   การทำความเข้าใจภาพ Spectrum ความถี่ของภาพ (Magnitude และ Phase Spectrum)
    *   ความแตกต่างในการกรองผ่านความถี่ต่ำ (Low-pass Filter - ภาพเบลอเนียน) และการกรองผ่านความถี่สูง (High-pass Filter - ตรวจหาขอบลายเส้น) ในโดเมนความถี่
    *   การแปลงฟูเรียร์กลับ (Inverse DFT) สู่ภาพต้นฉบับ
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 12: Filtering in Frequency Domain**:
    *   การคำนวณ DFT ของรูปภาพด้วยไลบรารี OpenCV และการย้ายศูนย์กลางสเปกตรัม (DFT Shift)
    *   การสร้างฟิลเตอร์รูปวงกลมความถี่ต่ำ (Ideal Low-pass filter, Butterworth Filter) ไปใช้งานกับภาพ
    *   การใช้ตัวกรองภาพลดสัญญาณรบกวนคลื่นความถี่สูงจำพวกคลื่นแทรกกวนรูปตารางลายตาสับปะรด (Periodic Noise Removal)
*   **เครื่องมือหลัก**: Python, OpenCV, NumPy, Matplotlib
*   **ฟังก์ชัน OpenCV ที่สำคัญ**: `cv2.dft()`, `cv2.idft()`, `cv2.magnitude()`
*   **งานปฏิบัติ**: เขียนโปรแกรมรับภาพที่มีแถบลายคลื่นสัญญาณรบกวนแบบสลับลายเส้นตรงแนวขวางแทรกกวนทั่วทั้งภาพ นำไปผ่านกระบวนการ DFT แล้วหาสัญญาณยอดแหลมที่ผิดปกติ (Spikes) ใน Magnitude Spectrum จากนั้นสร้าง Mask กรองค่าจุดเหล่านั้นออก ก่อนทำ Inverse DFT กลับมา เพื่อลบแถบลายรบกวนนั้นออกได้อย่างหมดจด

### สัปดาห์ที่ 13: การตรวจจับความเคลื่อนไหวและการติดตามวัตถุในวิดีโอ
*   **บรรยาย (2 ชั่วโมง)**:

    *   การตรวจหาความเคลื่อนไหวด้วยการเทียบความต่างเฟรม (Frame Differencing)
    *   อัลกอริทึมการหักลบภาพพื้นหลังแบบปรับตัวอัตโนมัติ (Adaptive Background Subtraction: MOG2 และ KNN Algorithms)
    *   ทฤษฎีการตรวจจับทิศทางการไหลเวียนแสง (Optical Flow): Lucas-Kanade (Sparse Optical Flow) และ Gunnar Farneback (Dense Optical Flow)
*   **ปฏิบัติการ (3 ชั่วโมง) - LAB 13: Motion Detection & Optical Flow Tracker**:
    *   เขียนโปรแกรมเปิดวิดีโอและรับภาพเรียลไทม์จากกล้อง CCTV
    *   การสร้างตัวตรวจจับขโมย/คนเดินผ่านกล้อง โดยตรวจจับความเคลื่อนไหว (Motion Detection System) ด้วย MOG2 และส่งการแจ้งเตือนเสียงหรือสกรีนภาพบันทึกพิกัด
    *   การนำภาพ Optical Flow มาวาดเส้นลูกศรแสดงทิศทางการขยับและเคลื่อนไหวของคนหน้ากล้อง

*   **งานปฏิบัติ**: ออกแบบระบบตรวจจับและแจ้งเตือนเมื่อรถยนต์ขับย้อนศร (Wrong-Way Driving Detection) โดยนำทฤษฎี Dense Optical Flow มาตรวจหาทิศทางเวกเตอร์การเคลื่อนไหวของวัตถุ ถ้าเวกเตอร์ชี้ไปในทิศตรงข้ามกับเลนที่กำหนด ให้ขีดกรอบสีแดงพร้อมตัวหนังสือแจ้งเตือนเตะตากลางเฟรม

*   **แนวคิด Mini-Project ที่สนับสนุน**: ดูรายละเอียดในหัวข้อถัดไป

---

### สัปดาห์ที่ 15: การทดสอบปลายภาคเรียน (Final Examination)
*   **การทดสอบเชิงทฤษฎี (2 ชั่วโมง)**:
    *   ข้อสอบวัดผลทฤษฎีขั้นสูง ครอบคลุมเรื่องระบบสีเชิงลึก, การสกัดคุณสมบัติ ORB/SIFT, การซ่อมแซมและบีบอัดภาพ, ทฤษฎีฟูเรียร์ในโดเมนความถี่ และขั้นตอนการวิเคราะห์การไหลของวัตถุวิดีโอ (Optical Flow)
*   **การทดสอบเชิงปฏิบัติการ (3 ชั่วโมง)**:
    *   สอบปฏิบัติเขียนโปรแกรมแบบปิดตำรา (Closed-book practical exam) เพื่อทดสอบความคล่องแคล่วและสมรรถนะในการแก้ไขปัญหาด้วยคำสั่งและฟังก์ชัน OpenCV
    *   *ตัวอย่างโจทย์*: ระบบตรวจจับหน้าปัดมาตรวัดกิโลวัตต์ชั่วโมง (Analog Dial/Gauge Reader): เขียนอัลกอริทึมหาตำแหน่งหน้าปัดวงกลม, ค้นหาเส้นตรงเข็มชี้วัดด้วย Hough Line, แล้วคำนวณมุมความชันของเข็มเทียบกับจุดศูนย์กลางเพื่อแปลงเป็นค่าตัวเลขระดับพลังงานปัจจุบัน

## 5. รายละเอียดของโครงงานย่อย (Mini-Project Guidelines)
เป้าหมายของ Mini-project คือการสร้างแอปพลิเคชันที่สามารถทำงานได้จริง โดยบูรณาการความรู้ที่เรียนตลอดภาคการศึกษาในการประมวลผลข้อมูลภาพและการนำมาประยุกต์ใช้งาน

### **แนวทางหัวข้อโครงงานที่แนะนำ (เลือก 1 หัวข้อ หรือเสนอใหม่):**
1.  **ระบบรู้จำแผ่นป้ายทะเบียนรถยนต์แบบจำกัดความเร็ว (Simple LPR System)**:
    *   *แนวคิด*: ตรวจจับแผ่นป้ายทะเบียนรถในภาพ ตัดภาพส่วนทะเบียนมาขยายความคมชัด (CLAHE + Morphological) ทำ Thresholding เพื่อเน้นตัวอักษร และตัดแยกตัวอักษรแต่ละตัวออกมาเก็บแยกโฟลเดอร์
2.  **เครื่องมือแต่งภาพและฟิลเตอร์อัจฉริยะแบบกึ่งอัตโนมัติ (Smart Photo Editor Desktop App)**:
    *   *แนวคิด*: สร้างโปรแกรม GUI (อาจใช้ Tkinter หรือ PyQT ร่วมกับ OpenCV) มีฟีเจอร์เด่น เช่น การลบรอยตำหนิใบหน้า (Inpainting), การแต่งแสงอัจฉริยะ, การใส่เอฟเฟกต์ภาพสเก็ตช์ภาพการ์ตูน และการยืดหดเปลี่ยนสัดส่วน
3.  **ระบบรักษาความปลอดภัยอัจฉริยะและบันทึกหน้าผู้บุกรุก (Smart Security Surveillance)**:
    *   *แนวคิด*: ใช้กล้อง Webcam คอยจับจ้องบริเวณที่หวงห้าม เมื่อมีคนผ่านกล้อง (Motion + Silhouette Detection) ให้ระบบทำการบันทึกภาพกรอบหน้าผู้สัญจร ส่งสัญญาณเสียงเตือน และจัดเก็บลงโฟลเดอร์แยกตามวันเวลา
4.  **ระบบคัดแยกผลไม้เกรดพรีเมียมจากมิติ สี และตำหนิ (Fruit Sorting & Grading Simulation)**:
    *   *แนวคิด*: วิเคราะห์ภาพผลไม้ผ่านสายพานจำลอง ตรวจวัดขนาดพื้นที่ (Contour Area) ตรวจเช็คความกลม (Circularity) ประเมินสีของผลเพื่อแยกความสุก (HSV Color Range) และค้นหารอยด่างดำหรือแผลบนเปลือกผลไม้
5.  **ระบบสแกนเอกสารและแปลงมุมภาพให้เรียบตรงอัตโนมัติ (Document Scanner with Perspective Transformation)**:
    *   *แนวคิด*: โปรแกรมวิเคราะห์ถ่ายรูปภาพเอกสารจากมุมเอียง ๆ ทำการค้นหาขอบเอกสารหลักสี่มุมด้วยโครงสร้างคอร์เนอร์ และทำการตบมุมบิดเบี้ยวให้เป็นมุมฉากตรง 90 องศา พร้อมทั้งปรับหน้ากระดาษให้ตัวอักษรดำคมชัดขึ้นเพื่อง่ายต่อการอ่าน
